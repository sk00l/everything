HTML Elements

A HTML element is everythong from start tag to closing tag.
<a> + element content +</a> is an element
<a> and <(opening)>(closing) are tags
They can be nested i.e. elements can contain other elements

HTML attributes

  • All HTML elements can have attributes
  • They provide more information about elements
  • they are always sepcified in starting tag.
  • example src="..." href="..."
  • elements have their own attributes

some attributes example


  • The above image is using element:
    <img src="image_url" height="200" width="200" alt="blackimage" />
  • Where for the img tag we need to provide an URL which can be absolute or relative.
  • The attritubes of an element can depend on the element it self.
  • Attributrs can be speicifed in four different ways
    1. Empty attribute syntax
      • Just the attribute name. The value is implicitly the empty string
      • <input disabled>
    2. Unquoted attribute value syntax
      • Just an attribute value with no " ' = < > `` and any spaces or tabs
      • Also cannot contain empty value
      • These works
        • <input value=yes>
      • These breaks
        • <input value=yes no>
          • space inside, not allowed
        • <input value=yes"no>
          • contains "
        • <input value=>
          • empty, not allowed
    3. Single-quoted attribute value syntax
      • Just an attribute whose value is wrapped in singl quote '
      • The value must start with ' and end with ' and inside the value it must not contain '
      • Example : <input type='checkbox'>
    4. Double-quoted attribute value syntax
      • Just an attribute whose value is wrapped in double quote ""
      • The value must start with " and end with " and inside the value it must not contain "
      • Example<input name="skool">

Iframe

  • Iframes are used where embedded contents is expected
  • the content attrbutes are:
    1. All global attributes
    2. src -- address of the resource -- if user controlled risk for phising, clickjacking
    3. srcdoc -- a document render in the iframe -- if user controlled rist for dirext XSS injection
    4. name -- name of content navigate
    5. sandbox -- secuiry rules for nested content
    6. allow -- permission polcy that can be applied to the iframe contents -- geolocation, camera, microphone, fullscreen, payments, autoplay
    7. allowfullscreen -- boolean -- lets iframe content use requestFullscreen()
    8. width -- horizontal dimension
    9. height -- vertical dimension
    10. referrerpolicy -- controls what referer header is send whtn iframe contents loads -- options are, no-referer, origin,strict-origin-when-cross-origin(default)
    11. loading -- tells browser whether to load iframe immediately or lazily -- lazy, eager, auto
  • Navigate to Iframe
  • Normally we use double quote for value of iframe attributes so, for srcdoc attribute if we need to use " and & we need to escape them by using, &quot; and &amp;
  • if both src and srcdoc is present the browser prefers srcdoc
